# Scaling Plans

Two plans for making the AI: Sand to Agents learning journey accessible to others.

## Current State
- 1.29MB single-file HTML on GitHub Pages
- localStorage for state, zero backend
- 107 concepts, 21 projects, 56 diagrams, 74 milestones, 411 hints

## Plans

### [Plan A: 10 Users](./plan-a-10-users.md)
- **Approach**: Keep static HTML + add optional Supabase sync
- **Cost**: $0/month (Supabase free tier)
- **Timeline**: 1-2 weekends
- **Key feature**: Cross-device sync via GitHub OAuth + cloud state

### [Plan B: 1 Million Users](./plan-b-1m-users.md)
- **Approach**: Full platform — Next.js + Neon + Clerk + Vercel
- **Cost**: $20/mo (1K users) → $3,970/mo (1M users)
- **Revenue**: $180K/mo at 1M users with 2% premium conversion
- **Timeline**: 12 months across 4 phases
- **Key features**: Accounts, spaced repetition, leaderboards, AI Q&A, certificates
